# Employee-Tracker

## Description

---

A CLI application for tracking your employees. Uses mysql database to view, add, or update your employees.

[Click here to watch a short video presentation of the application](https://drive.google.com/file/d/1wnTdHcNoofTczWfyLg3yg5q3wEcvDcPC/view)

## Installation

---

Run "npm i" to download all packages/dependencies and "node server.js" to start the application.

Schema and seeds available in repository.
